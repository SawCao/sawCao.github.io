阶段学习总结（一）

  1. DOCTYPE 意义

    !DOCTYPE> 声明必须是 HTML 文档的第一行，位于 html> 标签之前，指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令，在HTML5中，使用!DOCTYPE html>。

  2. 三种列表

    • ul是无序列表，没有排列顺序。
    • ol是有序列表，列表元素前有数字标识。
    • dl是定义列表，有前后层级关系。
    • 注：li为ul与ol的子元素，dt为dl的自定义列表项名，dd为其定义。

  1. 三列布局

    都是通过负边距 来实现左右两块固定大小的容器的位置确定。

    • 圣杯布局

      简述：通过container容器设置左右padding值，以及right,left容器设置左右负边距来确定位置。
  • 双飞翼布局

    简述：通过main容器中装入一个左右margin值的main_innner容器来实现与左右分离。

  1. 清除浮动

    目的：当两个元素，一个元素向左浮动，另一个元素向右浮动时，包裹它们的容器会发生高度塌陷，要解决这个问题。

    2. 精益求精的方法

  2. position

    1. 绝对定位absolute

      元素可以放在页面上任意位置，只需要指明其left,right,top,bottom值即可，绝对定位脱离于普通流。

    2. 相对定位relative

      相对于该元素正常位置进行偏移，指明其left,right,top,bottom值，即相对正常位置进行偏移。

    3. 固定定位（想对浏览窗口定位）fixed

      与相对定位一样，也是相对于该元素的正常位置进行偏移，但是偏移量是相对于整个窗口而不是原位置的。
