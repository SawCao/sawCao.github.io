CORS是一个W3C标准，全称是”跨域资源共享”（Cross-origin resource sharing）。

  1. 两类请求

    浏览器将CORS请求分成两类：简单请求（simple request）和非简单请求（not-so-simple request）。
    通常来说，除了 PUT 和 DELETE 请求以外，其他请求均为简单请求，
    本文只介绍简单请求的CORS。
